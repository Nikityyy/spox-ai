# SpoX+ AI


**BHAK & BHAS Steyr - Der intelligente Begleiter für die HAK Sport+**

![SpoX+ AI](httpdocs/public/assets/twitter.png)

SpoX+ AI ist eine moderne, KI-gestützte Web-App (Powered by Google Gemini), die im Rahmen einer **Diplomarbeit** an der **BHAK & BHAS Steyr** speziell für Schüler:innen der HAK Sport+ entwickelt wurde. Sie unterstützt bei der Matura-Vorbereitung, Lernorganisation, Training und Ernährung.

---

## Highlights & Features

- **Gemini**: Ultraschnelle Antworten und RAG (Retrieval Augmented Generation) für präzise, kontextbezogene Unterstützung basierend auf schulinternen Dokumenten.
- **Multimodal**: Unterstützung für Datei-Uploads (PDF, Bilder, Text), die direkt in den Chat einbezogen werden können.
- **Projekt-Management**: Organisiere deine Chats in Projekten und lade spezifische Dokumente für jedes Projekt hoch.
- **Microsoft Login**: Nahtlose Integration mit dem schulischen Microsoft-Konto (Entra ID) für maximale Sicherheit.
- **Made in Steyr**: Entwickelt von Schüler:innen für Schüler:innen der BHAK & BHAS Steyr.

---

## Technischer Stack

- **Frontend**: Vanilla HTML5/JS
- **Backend**: PHP API-Backend
- **Datenbank**: MySQL
- **KI**: Google Gemini API
